# 프로젝트 환경설정

git : 파이참 clone->  setting메뉴 선택후 project:pybo-project 클릭 
      add interpreter 클릭후 가상환경 구축 -> 터미널에서 가상환경 진입후 패키지 설치 진행
패키지 설치 : pip install -r requirements.txt
패키지 업로드 : pip freeze > requirements.txt
